# Proyecto-Master-DataScience-Evolve-CarmenAparicio
**Regenerative Agriculture Yield Analysis** 🌱

**Objetivo**  
- Analizar cómo variables relacionadas con: uso de fertilizantes, precipitaciones, temperatura, calidad del suelo y prácticas sostenibles que afectan al rendimiento de cultivos.
- Construir con esos datos un modelo simple de predicción.
